1. Всплывающее окно для ввода новых картинок (JS+css).
1. Всплывающее окно для вывода картинки на 80% экрана (JS+css).
4. Добавлен js код для запроса данных об авторе с сервера, пересылки данных об авторе на сервер и запроса картинок с сервера.
